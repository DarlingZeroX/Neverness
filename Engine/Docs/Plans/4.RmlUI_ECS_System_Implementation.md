## 架构定稿

```
┌─────────────────────────────────────────────────────────┐
│  NNRuntimeScene                                          │
│  ├─ NNRmlUIDocumentComponent (纯数据)                    │
│  ├─ IAssetResolver* (注入，不拥有)                        │
│  └─ NNRmlUISystem : ISceneSystem                        │
│       └─ Tick → 构建 RmlDrawList (纯数据，无 rendererHandle)│
└─────────────────────────────────────────────────────────┘
              ↓ RmlDrawList (只读)

┌─────────────────────────────────────────────────────────┐
│  ViewportRenderRuntimeApi (编排层)                        │
│  ├─ SceneRenderer → Sprite 纹理                          │
│  └─ RmlUIRenderer (持有 Rml::Context)                    │
│       ├─ SetViewport(width, height) — 每帧同步尺寸        │
│       ├─ Sync(drawList) — 加载/卸载文档 + 设置文档尺寸    │
│       ├─ RenderToTexture() — 渲染到内部 FBO，返回纹理 ID  │
│       │    ├─ BeginFrame() + Render() + EndFrameNoBlit() │
│       │    └─ 保存/恢复 ImGui GL state                    │
│       └─ 返回 RmlUI 纹理 ID                              │
│                                                          │
│  C# EditorViewport:                                      │
│  ├─ ImGui.Image(spriteTexture) — 底层 Sprite              │
│  └─ DrawList.AddImage(rmluiTexture) — 顶层 RmlUI 叠加    │
└─────────────────────────────────────────────────────────┘
```

---

## 三层职责边界（硬性约束）

| 层 | 持有 | 不持有 |
|----|------|--------|
| **NNRuntimeScene** | NNRmlUIDocumentComponent 数据 | Rml 任何东西 |
| **NNRmlUISystem** | EntityId → 组件状态映射 | Rml::ElementDocument / Context |
| **RmlUIRenderer** | Rml::Context + ElementDocument 实例 | ECS 组件 / PlayMode 状态 |
