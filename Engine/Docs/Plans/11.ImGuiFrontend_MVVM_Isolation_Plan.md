# ImGui 隔离实施计划：MVVM 架构 + ImGuiFrontend 模块

> **目标**：将 Editor 的 ImGui 具体实现隔离到独立模块 `Neverness.Editor.ImGuiFrontend`，
> 采用 **ViewModel + Controller + View** 架构，为未来 Avalonia 迁移做准备。
>
> **范围**：Phase 1（Framework 基础设施）+ ConsolePanel + SceneBrowser 作为首批验证对象。
>
> **日期**：2026-06-06

---

## 一、架构总览

### 1.1 目标架构（MVVM 三层分离）

```
Editor/
├── Neverness.Editor.Framework          ← UI 无关基础设施（IPanel, IMenuRegistry, Command 等）
├── Neverness.Editor.Core               ← ViewModel + Controller + 业务逻辑
│   ├── ViewModels/                     ← 纯数据 + 属性变更通知
│   │   ├── SceneBrowserViewModel.cs
│   │   ├── ContentBrowserViewModel.cs
│   │   └── ...
│   ├── Controllers/                    ← 业务操作（Rename/Delete/Create 等）
│   │   ├── SceneBrowserController.cs
│   │   └── ...
│   └── Services/                       ← 已有的 SceneManager, EventBus 等
│
├── Neverness.Editor.ImGuiFrontend      ← ImGui View 实现（唯一依赖 ImGui 的模块）
│   ├── Views/
│   │   ├── SceneBrowserImGuiView.cs
│   │   ├── ConsolePanelImGuiView.cs
│   │   └── ...
│   ├── Renderers/                      ← MenuRenderer, ToolbarRenderer 等
│   └── Infrastructure/                 ← ImGuiUIRenderer, ImGuiDockSpace 等
│
├── Neverness.Editor.Assets             ← 资产业务逻辑（不含 ImGui 渲染）
├── Neverness.Editor.Scene              ← 场景业务逻辑（不含 ImGui 渲染）
├── Neverness.Editor.Media              ← 媒体业务逻辑
└── NevernessEditor                     ← 入口，引用 ImGuiFrontend + 所有业务模块
```

### 1.2 依赖关系图（迁移后）

```
Level 0:  HexaGen.Runtime, Reflection, UndoRedo, ProjectSystem
Level 1:  Framework (UI-agnostic), ImGui (ThirdParty), ImGuiEx (-> ImGui)
Level 2:  Core (-> Framework, UndoRedo)                ← VM + Controller
          ImGuiFrontend (-> Framework, ImGuiEx, ImGui)  ← View 实现
Level 3:  Assets (-> Framework, Core)
          Scene  (-> Framework, Core)
Level 4:  Media, Rmlui, Script (-> Framework, Scene, Assets)
Level 5:  NevernessEditor (-> ImGuiFrontend + 所有业务模块)
```

**关键约束**：
- Framework **不再**依赖 ImGui / ImGuiEx
- Core **不再**依赖 ImGui / ImGuiEx
- Assets, Scene, Media, Script, Rmlui **不再**依赖 ImGui / ImGuiEx
- **只有** ImGuiFrontend 依赖 ImGui / ImGuiEx

### 1.3 数据流

```
用户操作 (ImGui 输入)
    ↓
ImGuiFrontend/View (渲染 ViewModel，捕获用户交互)
    ↓
Core/Controller (执行业务操作)
    ↓
Core/ViewModel (更新数据，触发 PropertyChanged)
    ↓
ImGuiFrontend/View (下一帧读取 ViewModel，刷新 UI)
```

```csharp
using System.ComponentModel;
using System.Runtime.CompilerServices;

namespace Neverness.Editor.Framework.Public.Mvvm;

/// <summary>
/// ViewModel 基类——提供属性变更通知的通用实现。
/// </summary>
public abstract class ViewModelBase : IViewModel
{
    public event Action<string>? PropertyChanged;

    public void OnPropertyChanged([CallerMemberName] string? propertyName = null)
    {
        PropertyChanged?.Invoke(propertyName ?? string.Empty);
    }

    /// <summary>设置属性值并触发变更通知。</summary>
    protected bool SetProperty<T>(ref T field, T value, [CallerMemberName] string? propertyName = null)
    {
        if (EqualityComparer<T>.Default.Equals(field, value))
            return false;

        field = value;
        OnPropertyChanged(propertyName);
        return true;
    }
}
```

```csharp
using Neverness.Editor.Framework.Public.Mvvm;

namespace Neverness.Editor.Core.ViewModels;

/// <summary>
/// 控制台 ViewModel——管理日志条目和过滤状态。
/// </summary>
public class ConsoleViewModel : ViewModelBase
{
    private readonly List<LogEntry> _entries = new();
    private string _filterText = "";
    private LogLevel _filterLevel = LogLevel.All;
    private bool _autoScroll = true;
    private int _selectedEntryIndex = -1;

    /// <summary>日志条目列表（只读视图）。</summary>
    public IReadOnlyList<LogEntry> Entries => _entries;

    /// <summary>过滤文本。</summary>
    public string FilterText
    {
        get => _filterText;
        set => SetProperty(ref _filterText, value);
    }

    /// <summary>过滤级别。</summary>
    public LogLevel FilterLevel
    {
        get => _filterLevel;
        set => SetProperty(ref _filterLevel, value);
    }

    /// <summary>是否自动滚动到底部。</summary>
    public bool AutoScroll
    {
        get => _autoScroll;
        set => SetProperty(ref _autoScroll, value);
    }

    /// <summary>选中的条目索引。</summary>
    public int SelectedEntryIndex
    {
        get => _selectedEntryIndex;
        set => SetProperty(ref _selectedEntryIndex, value);
    }

    /// <summary>过滤后的条目（View 层读取）。</summary>
    public IEnumerable<LogEntry> FilteredEntries =>
        _entries.Where(e => MatchesFilter(e));

    /// <summary>添加日志条目（Controller 调用）。</summary>
    public void AddEntry(LogEntry entry)
    {
        _entries.Add(entry);
        OnPropertyChanged(nameof(Entries));
        OnPropertyChanged(nameof(FilteredEntries));
    }

    /// <summary>清空日志（Controller 调用）。</summary>
    public void Clear()
    {
        _entries.Clear();
        OnPropertyChanged(nameof(Entries));
        OnPropertyChanged(nameof(FilteredEntries));
    }

    private bool MatchesFilter(LogEntry entry)
    {
        if ((entry.Level & _filterLevel) == 0) return false;
        if (!string.IsNullOrEmpty(_filterText) &&
            !entry.Message.Contains(_filterText, StringComparison.OrdinalIgnoreCase))
            return false;
        return true;
    }
}

public record LogEntry(string Message, LogLevel Level, DateTime Timestamp, string Category = "");

[Flags]
public enum LogLevel
{
    All = Debug | Info | Warning | Error | Fatal,
    Debug = 1,
    Info = 2,
    Warning = 4,
    Error = 8,
    Fatal = 16
}
```

### 3.4 新建 ImGui View

**文件**: `Engine/Source/Managed/Editor/Neverness.Editor.ImGuiFrontend/Views/ConsolePanelImGuiView.cs`

```csharp
using Hexa.NET.ImGui;
using Neverness.Editor.Core.ViewModels;
using Neverness.Editor.Framework.Public.Mvvm;

namespace Neverness.Editor.ImGuiFrontend.Views;

/// <summary>
/// 控制台 ImGui View——读取 ConsoleViewModel，渲染 ImGui UI。
/// </summary>
public class ConsolePanelImGuiView : PanelViewBase
{
    private ConsoleViewModel? _viewModel;

    public ConsolePanelImGuiView() : base("Console") { }

    public override Type ViewModelType => typeof(ConsoleViewModel);

    public override void Bind(object viewModel)
    {
        _viewModel = (ConsoleViewModel)viewModel;
        _viewModel.PropertyChanged += OnPropertyChanged;
    }

    public override void Unbind()
    {
        if (_viewModel != null)
            _viewModel.PropertyChanged -= OnPropertyChanged;
        _viewModel = null;
    }

    public override void Render()
    {
        if (_viewModel == null) return;

        if (!ImGui.Begin(GetWindowFullName()))
        {
            ImGui.End();
            return;
        }

        // 工具栏
        RenderToolbar();

        // 日志列表
        RenderLogList();

        ImGui.End();
    }

    private void RenderToolbar()
    {
        // 过滤按钮、清空按钮等
        if (ImGui.Button("Clear"))
        {
            // 通过 Controller 操作（此处简化为直接调用 ViewModel）
            _viewModel!.Clear();
        }
        ImGui.SameLine();
        // 过滤输入框
        var filterText = _viewModel!.FilterText;
        if (ImGui.InputText("Filter", ref filterText, 256))
            _viewModel.FilterText = filterText;
    }

    private void RenderLogList()
    {
        // 渲染过滤后的日志条目
        if (ImGui.BeginChild("LogRegion"))
        {
            foreach (var entry in _viewModel!.FilteredEntries)
            {
                ImGui.TextUnformatted($"[{entry.Level}] {entry.Message}");
            }

            if (_viewModel.AutoScroll)
                ImGui.SetScrollHereY(1.0f);

            ImGui.EndChild();
        }
    }

    private void OnPropertyChanged(string propertyName)
    {
        // ViewModel 属性变更时的处理（如果需要即时响应）
    }
}
```

```csharp
    /// <summary>根节点列表（View 读取用于渲染）。</summary>
    public IReadOnlyList<EntityNodeVM> RootNodes =>
        string.IsNullOrEmpty(_searchText) ? _rootNodes : _filteredNodes;

    /// <summary>所有节点（扁平化，用于搜索过滤）。</summary>
    public IReadOnlyList<EntityNodeVM> AllNodes => _allNodes;

    /// <summary>当前选中的实体 Handle。</summary>
    public ulong SelectedEntityHandle
    {
        get => _selectedEntityHandle;
        set => SetProperty(ref _selectedEntityHandle, value);
    }

    /// <summary>多选的实体 Handle 集合。</summary>
    public IReadOnlySet<ulong> MultiSelectedHandles => _multiSelectedHandles;

    /// <summary>搜索过滤文本。</summary>
    public string SearchText
    {
        get => _searchText;
        set
        {
            if (SetProperty(ref _searchText, value))
                UpdateFilteredNodes();
        }
    }

    /// <summary>拖拽源实体 Handle。</summary>
    public ulong DragSourceHandle
    {
        get => _dragSourceHandle;
        set => SetProperty(ref _dragSourceHandle, value);
    }

    /// <summary>拖拽目标实体 Handle。</summary>
    public ulong DragTargetHandle
    {
        get => _dragTargetHandle;
        set => SetProperty(ref _dragTargetHandle, value);
    }

    // ── 方法（Controller 调用） ──

    /// <summary>重建实体节点树（场景加载/变更时调用）。</summary>
    public void RebuildTree(List<EntityNodeVM> rootNodes, List<EntityNodeVM> allNodes)
    {
        _rootNodes = rootNodes;
        _allNodes = allNodes;
        UpdateFilteredNodes();
        OnPropertyChanged(nameof(RootNodes));
        OnPropertyChanged(nameof(AllNodes));
    }

    /// <summary>设置实体展开状态。</summary>
    public void SetExpanded(ulong handle, bool expanded)
    {
        if (expanded) _expandedHandles.Add(handle);
        else _expandedHandles.Remove(handle);
    }

    /// <summary>检查实体是否展开。</summary>
    public bool IsExpanded(ulong handle) => _expandedHandles.Contains(handle);

    /// <summary>设置选中实体。</summary>
    public void Select(ulong handle)
    {
        _selectedEntityHandle = handle;
        OnPropertyChanged(nameof(SelectedEntityHandle));
    }

    /// <summary>添加到多选。</summary>
    public void AddToSelection(ulong handle)
    {
        _multiSelectedHandles.Add(handle);
        OnPropertyChanged(nameof(MultiSelectedHandles));
    }

    /// <summary>清空选择。</summary>
    public void ClearSelection()
    {
        _selectedEntityHandle = 0;
        _multiSelectedHandles.Clear();
        OnPropertyChanged(nameof(SelectedEntityHandle));
        OnPropertyChanged(nameof(MultiSelectedHandles));
    }

    private void UpdateFilteredNodes()
    {
        _filteredNodes = _allNodes
            .Where(n => n.Name.Contains(_searchText, StringComparison.OrdinalIgnoreCase))
            .ToList();
    }
}

/// <summary>
/// 实体节点 ViewModel——单个实体在树中的表示。
/// </summary>
public class EntityNodeVM
{
    public ulong Handle { get; init; }
    public string Name { get; set; } = "";
    public ulong ParentHandle { get; init; }
    public List<EntityNodeVM> Children { get; init; } = new();
    public bool IsRoot => ParentHandle == 0;
    public int Depth { get; init; }
    public bool HasChildren => Children.Count > 0;
}
```

```csharp
using Hexa.NET.ImGui;
using Neverness.Editor.Core.ViewModels;
using Neverness.Editor.Framework.Public;
using Neverness.Editor.Framework.Public.Mvvm;

namespace Neverness.Editor.ImGuiFrontend.Views;

/// <summary>
/// 场景浏览器 ImGui View——读取 SceneBrowserViewModel，渲染 ImGui 树形 UI。
/// </summary>
public class SceneBrowserImGuiView : PanelViewBase
{
    private SceneBrowserViewModel? _viewModel;
    private SceneBrowserController? _controller;
    private readonly IAssetDragDropService _dragDropService;
    private readonly IContextMenuRenderer _contextMenuRenderer;
    private readonly ContextMenuManager _contextMenuManager;

    // 缓存：每帧重建太慢，用增量更新
    private float _rowHeight;
    private const string CONTEXT_ID = "SceneBrowser.Context";

    public SceneBrowserImGuiView(
        IAssetDragDropService dragDropService,
        IContextMenuRenderer contextMenuRenderer,
        ContextMenuManager contextMenuManager) : base("Scene Browser")
    {
        _dragDropService = dragDropService;
        _contextMenuRenderer = contextMenuRenderer;
        _contextMenuManager = contextMenuManager;
    }

    public override Type ViewModelType => typeof(SceneBrowserViewModel);

    public override void Bind(object viewModel)
    {
        _viewModel = (SceneBrowserViewModel)viewModel;
        // Controller 通过服务定位器获取
    }

    public override void Unbind()
    {
        _viewModel = null;
    }

    public override void Render()
    {
        if (_viewModel == null) return;

        if (!ImGui.Begin(GetWindowFullName()))
        {
            ImGui.End();
            return;
        }

        RenderToolbar();
        RenderSearchBar();
        RenderEntityTree();
        RenderContextMenu();

        ImGui.End();
    }

    // ── 工具栏 ──
    private void RenderToolbar()
    {
        if (ImGui.Button("Refresh"))
            _controller?.RefreshTree();
        ImGui.SameLine();
        if (ImGui.Button("Create"))
            _controller?.CreateChildEntity(0);
    }

    // ── 搜索栏 ──
    private void RenderSearchBar()
    {
        var searchText = _viewModel!.SearchText;
        if (ImGui.InputTextWithHint("##Search", "Search entities...", ref searchText, 256))
            _controller?.SetSearchText(searchText);
        ImGui.Separator();
    }

    // ── 实体树 ──
    private void RenderEntityTree()
    {
        if (ImGui.BeginChild("TreeRegion"))
        {
            _rowHeight = ImGui.GetFrameHeightWithSpacing();

            foreach (var node in _viewModel!.RootNodes)
            {
                RenderEntityNode(node);
            }

            ImGui.EndChild();
        }
    }

    private void RenderEntityNode(EntityNodeVM node)
    {
        var flags = ImGuiTreeNodeFlags.OpenOnArrow | ImGuiTreeNodeFlags.OpenOnDoubleClick;

        if (!node.HasChildren)
            flags |= ImGuiTreeNodeFlags.Leaf | ImGuiTreeNodeFlags.NoTreePushOnOpen;

        if (node.Handle == _viewModel!.SelectedEntityHandle)
            flags |= ImGuiTreeNodeFlags.Selected;

        var expanded = _viewModel.IsExpanded(node.Handle);

        // 设置展开状态
        ImGui.SetNextItemOpen(expanded);

        // 渲染树节点
        bool opened = ImGui.TreeNodeEx(
            $"{node.Name}##{node.Handle}",
            flags);

        // 更新展开状态
        if (node.HasChildren)
            _viewModel.SetExpanded(node.Handle, opened);

        // 点击选中
        if (ImGui.IsItemClicked())
            _controller?.SelectEntity(node.Handle);

        // 拖拽源
        if (ImGui.BeginDragDropSource())
        {
            ImGui.SetDragDropPayload("ENTITY_HANDLE", node.Handle);
            ImGui.Text(node.Name);
            ImGui.EndDragDropSource();
        }

        // 拖拽目标
        if (ImGui.BeginDragDropTarget())
        {
            if (ImGui.AcceptDragDropPayload("ENTITY_HANDLE").Data != IntPtr.Zero)
            {
                _controller?.ReparentEntity(_viewModel.DragSourceHandle, node.Handle);
            }
            ImGui.EndDragDropTarget();
        }

        // 右键菜单
        if (ImGui.BeginPopupContextItem($"##ctx_{node.Handle}"))
        {
            if (ImGui.MenuItem("Rename"))
            {
                // 弹出重命名对话框
            }
            if (ImGui.MenuItem("Delete"))
            {
                _controller?.DeleteEntity(node.Handle);
            }
            if (ImGui.MenuItem("Create Child"))
            {
                _controller?.CreateChildEntity(node.Handle);
            }
            ImGui.EndPopup();
        }

        // 渲染子节点
        if (opened && node.HasChildren)
        {
            foreach (var child in node.Children)
            {
                RenderEntityNode(child);
            }
            ImGui.TreePop();
        }
    }

    // ── 上下文菜单（空白区域） ──
    private void RenderContextMenu()
    {
        _contextMenuRenderer.RenderWindowContextMenu(CONTEXT_ID, _contextMenuManager);
    }
}
```

## 十一、FAQ

**Q: 为什么不直接用 Avalonia 的 MVVM 框架（如 ReactiveUI / CommunityToolkit.Mvvm）？**
A: 当前目标是验证架构可行性，不引入额外依赖。ViewModelBase 已提供足够基础。
未来可选择迁移到 CommunityToolkit.Mvvm 的 ObservableObject。

**Q: IView.Bind(object) 为什么用 object 而不是泛型？**
A: 为了统一 PanelViewBase 基类。具体 View 内部会做类型转换。
如果后续需要更强类型安全，可改为 `IView<TViewModel>`。

**Q: Controller 如何访问 SceneNativeBridge 等运行时服务？**
A: 通过 `IEditorContext.GetService<T>()` 服务定位器。
Controller 在 Initialize() 时获取所需服务的引用。

**Q: 搬迁后旧的 IEditorPanel 接口还保留吗？**
A: 保留。PanelViewBase 同时实现 IEditorPanel 和 IView，
PanelManager 无需修改即可管理新旧面板。未来可逐步废弃 IEditorPanel。
