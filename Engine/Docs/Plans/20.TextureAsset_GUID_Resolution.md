# TextureAsset GUID → GPU Handle 解析计划

## 1. 问题

`ViewportServiceImpl.CollectRenderCommands()` 第 269 行：
```csharp
instance.TextureHandle = 0; // TODO: 从 TextureAsset GUID 解析 GPU Handle
```

所有精灵的 `TextureHandle` 始终为 0，C++ 端只能使用白色默认纹理。

## 2. C++ 端参考实现

```
SpriteRendererComponent.TextureAsset (NNGuid)
    ↓ SpriteRenderSystem::Collect()
RenderAssetManager.GetCacheKeyByGuidLow(guidLow)
    ↓ cache miss?
AssetManager.LoadAssetSync(guid) → GetBlobByType(TypeInfo/Data)
RenderAssetManager.LoadTextureFromBlob(typeInfo, pixelData, guidLow)
    ↓
TextureResource.GetImGuiHandle() → reinterpret_cast<uint64_t>(ITextureView*)
    ↓
SpriteInstance.TextureHandle = handle
```

1. `TextureResource.GetImGuiHandle()` 是 stub，返回 0
2. `ViewportServiceImpl.CollectRenderCommands()` 没有从 GUID 解析 Handle

## 4. 实施计划

### Phase 1：修复 TextureResource.GetImGuiHandle()

**文件**: `Neverness.Rendering.RenderAssets/TextureResource.cs`

`TextureView` 内部持有 `ITextureView NativeObject`。`ITextureView` 继承自 `ComObject` → `CppObject`，有 `NativePointer` 属性（`IntPtr`）。

**方案 A（推荐）**: 在 `TextureView` 上暴露原生指针：

```csharp
// TextureView.cs — Neverness.Rendering.Diligent
internal IntPtr NativePointer => NativeObject?.NativePointer ?? IntPtr.Zero;
```

然后 `TextureResource.GetImGuiHandle()` 改为：

```csharp
public ulong GetImGuiHandle()
{
    if (_rhiShaderResourceView == null)
        return 0;
    return (ulong)_rhiShaderResourceView.NativePointer;
}
```

**已确认**: `ComObject` 继承自 `CppObject`，`NativePointer` 属性可用。与 C++ 端 `reinterpret_cast<uint64_t>(ITextureView*)` 等价。

### Phase 2：RenderAssetManager 增加 EnsureTextureLoaded 方法

**文件**: `Neverness.Rendering.RenderAssets/RenderAssetManager.cs`

新增一个完整的 GUID → GPU Handle 解析方法，封装全链路（缓存查询 → 资产加载 → Blob 解析 → GPU 上传 → 返回 Handle）：

```csharp
/// <summary>
/// 确保纹理已加载到 GPU，返回 TextureHandle（ITextureView* 编码为 ulong）。
/// 内部完成：缓存查询 → AssetManager.LoadAssetSync → Blob 解析 → GPU 上传。
/// 返回 0 表示失败（空 GUID、加载失败、上传失败）。
/// </summary>
public ulong EnsureTextureLoaded(NNGuid textureGuid)
{
    // 1. 空 GUID 检查
    if (textureGuid.High == 0 && textureGuid.Low == 0)
        return 0;

    // 2. 查询缓存
    var cacheKey = GetCacheKeyByGuidLow(textureGuid.Low);
    if (cacheKey != 0)
        return GetGLTextureId(cacheKey);

    // 3. Cache miss → 从 AssetManager 加载
    var guid = new GUID(textureGuid.High, textureGuid.Low);
    var handle = AssetManager.Instance.LoadAssetSync(guid);
    if (handle == 0)
        return 0;

    // 4. 解析 Blob（TypeInfo + Data）
    var typeInfo = AssetManager.Instance.GetBlobByType(handle, BlobType.TypeInfo);
    var pixelData = AssetManager.Instance.GetBlobByType(handle, BlobType.Data);
    if (typeInfo.IsEmpty || pixelData.IsEmpty)
        return 0;

    // 5. 上传 GPU
    unsafe
    {
        fixed (byte* typeInfoPtr = typeInfo)
        fixed (byte* pixelPtr = pixelData)
        {
            cacheKey = LoadTextureFromBlob(
                (IntPtr)typeInfoPtr, (ulong)typeInfo.Length,
                (IntPtr)pixelPtr, (ulong)pixelData.Length,
                textureGuid.Low);
        }
    }

    return cacheKey != 0 ? GetGLTextureId(cacheKey) : 0;
}
```

**依赖变更**: `Neverness.Rendering.RenderAssets.csproj` 需新增 `Runtime.Assets` 引用（用于 `AssetManager`、`GUID`、`BlobType`）。
