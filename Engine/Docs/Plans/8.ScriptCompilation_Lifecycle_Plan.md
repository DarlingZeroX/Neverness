## 1. 架构总览

```
编译管线：
  .cs → ScriptCompileQueue → ScriptCompileService → ScriptCompiler → UserScripts.dll
  → ScriptAssemblyLoader.Load() → 扫描类型 → ScriptRegistry.Replace()

运行时管线：
  ScriptComponent (ECS)
        ↓
  ScriptBehaviourBridge (桥接，操作 BehaviourRegistry)
        ↓
  IScriptFactory (创建抽象)
        ↓
  BehaviourRegistry (存储)
        ↓
  ScriptBehaviourScheduler (只负责 Tick)

热重载管线：
  Compile → Unload ALC → Load ALC → Registry Replace
  → MarkForDestroy 所有 Behaviour → Bridge 下一帧自动重建
```

---

## 2. 关键规则

| 规则 | 说明 |
|------|------|
| ECS 是唯一权威 | ScriptComponent 存 ScriptTypeId + Enabled |
| Bridge 只操作 BehaviourRegistry | 不依赖 Scheduler 查询 |
| Scheduler 只负责 Tick | 不负责创建、不负责存储查询 |
| IScriptFactory 抽象创建 | 当前 Activator.CreateInstance，未来 DI |
| 热重载用 MarkForDestroy | 不用 Clear()，保证 OnDestroy 执行 |
| Registry 用 Replace | 不用 Clear()，TypeId 稳定，避免空窗期 |
| ScriptAssetIndex 事件驱动 | 监听 CompileFinished 事件，不在 CompileService 中直接调用 |
| AutoRegisterScript 删除 | 所有 EntityBehaviour 子类自动注册 |
| Bridge 最终事件驱动 | Phase 1 全量扫描，最终改 entt on_construct/on_update/on_destroy |

---

## 3. 生命周期顺序

```
Frame N:   Bridge 创建 Behaviour → BehaviourRegistry.Register() → 加入 pendingCreate
Frame N:   Scheduler.Tick()
             1. ProcessPendingCreate() → OnCreate()
             2. ProcessPendingStart()  → OnStart() (上一帧创建的)
             3. foreach active: OnUpdate(deltaTime)
             4. ProcessPendingDestroy() → OnDestroy()
Frame N+1: OnStart() 对 Frame N 创建的 Behaviour 调用
```

### 4.3 ScriptBehaviourBridge

```csharp
/// <summary>
/// 脚本行为桥接层——桥接 ECS ScriptComponent 与 Behaviour 实例。
///
/// 职责：
/// - 发现有 ScriptComponent 但无 Behaviour → 创建
/// - 发现 ScriptTypeId 变化 → 销毁旧 + 创建新
/// - 发现 ScriptComponent 移除 → 销毁
/// - 同步 Enabled 状态
///
/// 不依赖 Scheduler 查询，只操作 BehaviourRegistry。
///
/// Phase 1: 每帧全量扫描 Query&lt;ScriptComponent&gt;
/// Phase 2: 改为 entt on_construct / on_update / on_destroy 事件驱动
/// </summary>
public sealed class ScriptBehaviourBridge
{
    private readonly IScriptFactory _factory;
    private readonly BehaviourRegistry _registry;
    private readonly ScriptRegistry _scriptRegistry;
    private readonly ScriptBehaviourScheduler _scheduler;

    /// <summary>上一帧有 ScriptComponent 的 Entity 集合（用于检测 Component 删除）。</summary>
    private readonly HashSet<ulong> _previousScriptEntities = new();

    public void Sync(SceneWorld world)
    {
        var currentEntities = new HashSet<ulong>();

        foreach (var entity in world.Query<ScriptComponent>())
        {
            currentEntities.Add(entity.Id);
            var script = entity.GetComponent<ScriptComponent>();

            // 直接查 BehaviourRegistry，不经过 Scheduler
            var existing = _registry.GetBehaviours(entity).FirstOrDefault();
        {
            var script = entity.GetComponent<ScriptComponent>();

            // 直接查 BehaviourRegistry，不经过 Scheduler
            var existing = _registry.GetBehaviours(entity).FirstOrDefault();

            if (script.ScriptTypeId == 0)
            {
                // 无脚本 → 销毁 Behaviour
                if (existing != null)
                    _scheduler.MarkForDestroy(existing);
                continue;
            }

            var scriptInfo = _scriptRegistry.FindByTypeId(script.ScriptTypeId);
            if (scriptInfo == null)
                continue;

            if (existing == null)
            {
                // 创建
                var behaviour = _factory.Create(scriptInfo.Type);
                if (behaviour != null)
                {
                    behaviour.Entity = entity;
                    behaviour.Enabled = script.Enabled;
                    _scheduler.Register(entity, behaviour);
                }
            }
            else if (existing.GetType() != scriptInfo.Type)
            {
                // 类型变化 → 销毁旧 + 创建新
                _scheduler.MarkForDestroy(existing);
                var behaviour = _factory.Create(scriptInfo.Type);
                if (behaviour != null)
                {
                    behaviour.Entity = entity;
                    behaviour.Enabled = script.Enabled;
                    _scheduler.Register(entity, behaviour);
                }
            }
            else
            {
                existing.Enabled = script.Enabled;
            }
        }

        // 检测 ScriptComponent 被删除的 Entity（上一帧有，这一帧没有）
        foreach (var entityId in _previousScriptEntities)
        {
            if (!currentEntities.Contains(entityId))
            {
                // ScriptComponent 被移除 → 销毁 Behaviour
                // 需要通过 entityId 查找 Behaviour（Bridge 持有 BehaviourRegistry 引用）
                // BehaviourRegistry 提供 Entity → Behaviour 映射
            }
        }

        _previousScriptEntities.Clear();
        foreach (var id in currentEntities)
            _previousScriptEntities.Add(id);
    }
}
```

### 4.4 ScriptCompileService

```csharp
/// <summary>
/// 脚本编译服务——编排 编译→加载→注册 完整流程。
///
/// 职责边界：
/// - 编译、加载、类型注册
/// - 触发 CompileFinished 事件
/// - 不直接操作 ScriptAssetIndex（由事件驱动）
/// </summary>
public sealed class ScriptCompileService
{
    private readonly ScriptCompiler _compiler = new();
    private readonly ScriptAssemblyLoader _assemblyLoader = new();

    /// <summary>编译完成事件。</summary>
    public event EventHandler<ScriptCompileResult>? CompileFinished;

    /// <summary>
    /// 编译 + 加载 + 注册。
    /// </summary>
    public ScriptCompileResult CompilePipeline(string projectDirectory)
    {
        // 1. 编译
        var options = ScriptCompileOptions.CreateDefault();
        options.ScriptRoot = Path.Combine(projectDirectory, "Assets");
        options.OutputPath = Path.Combine(projectDirectory, "Library", "Scripts");
        var result = _compiler.Compile(options);
        if (!result.Success)
            return ToEditorResult(result);

        // 2. 加载程序集
        _assemblyLoader.Load(result.OutputPath!);

        // 3. 扫描并注册类型（Replace，不用 Clear）
        RegisterTypesFromAssembly(_assemblyLoader.LoadedAssembly!);

        var editorResult = ToEditorResult(result);
        CompileFinished?.Invoke(this, editorResult);
        return editorResult;
    }

    /// <summary>
    /// 热重载：销毁 Behaviour → FlushDestroy → 卸载 ALC → 加载新 ALC → Replace Registry。
    /// 下一帧 Bridge 自动从 ECS 重建。
    ///
    /// 顺序很重要：
    /// 1. MarkForDestroy + FlushDestroy（OnDestroy 在旧 ALC 中执行）
    /// 2. Unload 旧 ALC
    /// 3. Load 新 ALC
    /// 4. Registry Replace（新 Type 覆盖旧 TypeId）
    /// </summary>
    public ScriptCompileResult HotReload(string projectDirectory)
    {
        var scheduler = GameplayContext.Current?.BehaviourScheduler;
        var behaviourRegistry = GameplayContext.Current?.BehaviourRegistry;

        // 1. MarkForDestroy 所有 Behaviour + 立即 FlushDestroy（OnDestroy 在旧 ALC 中执行）
        if (scheduler != null && behaviourRegistry != null)
        {
            foreach (var behaviour in behaviourRegistry.AllBehaviours.ToList())
            {
                scheduler.MarkForDestroy(behaviour);
            }
            behaviourRegistry.ProcessPendingDestroy();
        }

        // 2. 编译 + 卸载旧 ALC + 加载新 ALC + Registry Replace
        var result = CompilePipeline(projectDirectory);

        return result;
    }

    /// <summary>
    /// 扫描程序集，注册 EntityBehaviour 子类到 ScriptRegistry。
    /// 用 Replace 替代 Clear，TypeId 稳定，避免空窗期。
    /// </summary>
    private void RegisterTypesFromAssembly(Assembly assembly)
    {
        var registry = GameplayContext.Current?.ScriptRegistry;
        if (registry == null) return;

        foreach (var type in assembly.GetTypes())
        {
            if (type.IsAbstract || type.IsInterface) continue;
            if (!typeof(EntityBehaviour).IsAssignableFrom(type)) continue;

            // 直接注册，不需要 [AutoRegisterScript]
            registry.Register(type);
        }
    }
}
```

### 4.5 ScriptEditorServiceImpl 修改

```csharp
public sealed class ScriptEditorServiceImpl : IScriptEditorService, IDisposable
{
    private readonly ScriptCompileService _compileService = new();
    private string? _projectDirectory;

    public ScriptEditorServiceImpl()
    {
        ScriptCompileQueue.CompileTriggered += OnCompileTriggered;
    }

    private async void OnCompileTriggered(object? sender, IReadOnlyList<string> files)
    {
        if (_projectDirectory == null) return;
        await CompileAllAsync();
    }

    public async Task<ScriptCompileResult> CompileAllAsync(CancellationToken ct = default)
    {
        if (_projectDirectory == null)
            return new ScriptCompileResult { Status = ScriptCompileStatus.Error };

        Status = ScriptCompileStatus.Compiling;

        var result = await Task.Run(() =>
            _compileService.CompilePipeline(_projectDirectory), ct);

        LastResult = result;
        Status = result.Status;
        CompileFinished?.Invoke(this, result);
        return result;
    }

    public async Task<bool> RequestHotReloadAsync(CancellationToken ct = default)
    {
        if (_projectDirectory == null) return false;

        var result = await Task.Run(() =>
            _compileService.HotReload(_projectDirectory), ct);

        LastResult = result;
        return result.Success;
    }
}
```

## 8. 风险

1. **dotnet build 耗时**：首次编译几秒（NuGet 还原），增量编译较快。

2. **ALC 卸载非阻塞**：Unload() 后 GC 回收时机不确定，确保旧引用已释放。

3. **Bridge Phase 1 全量扫描**：10000 Entity 中 200 Script → 可接受。Phase 5 改事件驱动后消除。

4. **Behaviour 状态丢失**：热重载后 Behaviour 实例全新，Inspector 字段丢失。Editor JIT 固有限制。

5. **编译输出目录**：`Library/Scripts/` 不在 Assets 下，避免 AssetWatcher 循环编译。

6. **ScriptRegistry.Replace 语义**：现有 Register() 已支持覆盖（Dictionary key），无需新增 API。
