## 概述

将 RmlUI 文档组件从 C++ Legacy 系统移植到新 ECS 系统。

设计原则：
- **序列化数据与运行时状态分离** — RuntimeHandle 等瞬态数据不参与序列化
- **Flags 用 enum** — 不用裸 uint32，Inspector 自动支持 Checkbox
- **Inspector 显示资产名** — 用户看到 `MainMenu.html` 而不是 GUID
- **菜单分类清晰** — UI 类实体归入 "Add Entity/UI/" 子菜单
