# Inspector 不显示组件修复

日期：2026-06-17

---

## 根因

`InspectorController.RefreshComponents()` 是 TODO 空壳——永远返回空列表，从未调用 `IInspectorService.GetEntityComponents()`。

```csharp
// 旧代码（错误）
public void RefreshComponents()
{
    if (!_viewModel.HasSelection) return;

    // TODO: 通过 entityId 获取 IEntity 实例
    // 暂时返回空列表
    var components = new List<ComponentDataInfo>();  // ← 永远空！
    ...
}
```

此外，`ComponentInspectorRegistry` 在 Avalonia 模式下为空——只有 `ImGuiFrontendModule.Install()` 会扫描注册 `IComponentInspector` 实现，而 Avalonia 模式不加载 ImGuiFrontend。

---

## 修复

### 1. InspectorController.RefreshComponents() — 真正实现

**文件**: `Engine/Source/Managed/Editor/Neverness.Editor.Core/Controllers/InspectorController.cs`

```csharp
public void RefreshComponents()
{
    if (!_viewModel.HasSelection) return;

    // 通过 entityId 获取 IEntity 实例
    var entity = _inspectorService.GetEntityById(_viewModel.SelectedEntityId);
    if (entity == null || !entity.IsValid)
    {
        _viewModel.UpdateComponents(new List<ComponentInfoVM>());
        return;
    }

    // 从 Service 获取组件列表
    var components = _inspectorService.GetEntityComponents(entity);

    var vmComponents = components.Select(c => new ComponentInfoVM
    {
        TypeId = c.TypeId,
        DisplayName = c.DisplayName,
        Order = c.Order
    }).ToList();

    _viewModel.UpdateComponents(vmComponents);
}
```

### 2. IInspectorService + InspectorServiceImpl — 新增 GetEntityById

**文件**: `Engine/Source/Managed/Editor/Neverness.Editor.Core/Public/IInspectorService.cs`

```csharp
/// <summary>按 ID 获取实体实例（返回 null 表示不存在）。</summary>
IEntity? GetEntityById(int entityId);
```

**文件**: `Engine/Source/Managed/Editor/Neverness.Editor.Scene/Private/Service/InspectorServiceImpl.cs`

```csharp
public IEntity? GetEntityById(int entityId)
{
    var scene = SceneModuleImp.SceneQueryService?.ActiveScene;
    return scene?.GetEntity(entityId);
}
```

### 3. GenericComponentInspector — 轻量 IComponentInspector 实现

**文件**: `Engine/Source/Managed/Editor/Neverness.Editor.Scene/Private/Inspector/GenericComponentInspector.cs`（新建）

仅实现 `HasComponent` / `RemoveComponent` / `TypeId`，不绘制 UI。Avalonia 前端由 `AvaloniaComponentInspectorRegistry` 负责绘制。

```csharp
internal sealed class GenericComponentInspector<T> : IComponentInspector where T : struct, IComponent
{
    private readonly ulong _typeId;
    private readonly string _displayName;
    private readonly int _order;

    public GenericComponentInspector(ulong typeId, string displayName, int order = 1000) { ... }

    public bool HasComponent(IEntity entity) => entity.Has<T>();
    public void RemoveComponent(IEntity entity) => entity.Remove<T>();
    public bool DrawInspector(IEntity entity) => false; // 不绘制 UI
}
```

### 4. SceneModuleImp — 注册通用检查器

**文件**: `Engine/Source/Managed/Editor/Neverness.Editor.Scene/Private/SceneModuleImp.cs`

在 `Install()` 中调用 `RegisterGenericInspectors()`，注册 Transform/Camera/SpriteRenderer：

```csharp
private static void RegisterGenericInspectors()
{
    ComponentInspectorRegistry.Register(new GenericComponentInspector<TransformComponent>(
        0xC1FFF4F356DFB2FB, "Transform", order: 0));
    ComponentInspectorRegistry.Register(new GenericComponentInspector<CameraComponent>(
        0x54D1B2A64667E32E, "Camera", order: 10));
    ComponentInspectorRegistry.Register(new GenericComponentInspector<SpriteRendererComponent>(
        0x51387BA3968C343B, "SpriteRenderer", order: 20));
}
```

> 注意：TypeId 须与 Native `NN_REGISTER_COMPONENT` 的 FNV-1a hash 一致。
> Script/AudioSource/VideoPlayer/RmlUIDocument 的 TypeId 由 `RmluiModule` 和 `MediaModule` 注册。

**文件**: `Engine/Source/Managed/Editor/Neverness.Editor.AvaloniaFrontend/Views/InspectorAvaloniaView.cs`

## 注册来源

`ComponentInspectorRegistry` 的检查器来自三个模块：

| 模块 | 注册的组件 | 方式 |
|------|-----------|------|
| `SceneModuleImp` | Transform, Camera, SpriteRenderer | `RegisterGenericInspectors()` |
| `RmluiModule` | RmlUIDocument | `DiscoverFromAssembly()` |
| `MediaModule` | AudioSource, VideoPlayer | `DiscoverFromAssembly()` |

ImGuiFrontend 模式下额外注册所有组件的完整 Inspector（含 UI 绘制）。

## 数据流（修复后）

```
用户点击 Entity
→ SceneBrowserViewModel.Select(handle)
→ SelectedEntityId 变更 → PropertyChanged
→ InspectorController.RefreshComponents()
  → InspectorService.GetEntityById(id) → IEntity
  → InspectorService.GetEntityComponents(entity)
    → ComponentInspectorRegistry 遍历
    → GenericComponentInspector.Has<T>() → true/false
    → 返回 List<ComponentDataInfo>
  → ViewModel.UpdateComponents(components)
→ PropertyChanged("Components")
→ InspectorAvaloniaView.RebuildComponentList() (UI 线程)
  → AvaloniaComponentInspectorRegistry.CreateInspectorUI()
  → 渲染组件 Inspector UI
```
