# SceneBrowser 上下文菜单扩展架构

> **状态：** 已完成
>
> **范围：** 通过 ContextMenuManager 实现 SceneBrowser "Add Entity" 菜单的跨模块扩展

---

## 架构概览

```
SceneBrowser.DrawBackgroundContextMenu()
    ↓
ContextMenuManager.RenderWindowContextMenu("scene_browser.background")
    ↓
├── SceneBrowserContextMenuContributor (Camera, Sprite)
└── MediaContextMenuContributor (Audio Source, Video Player)
```

## 设计原则

1. **不硬编码** — SceneBrowser 本身不包含任何菜单项，全部通过 ContextMenuManager 扩展
2. **跨模块注册** — 任何模块都可以通过 `EditorMenuRegistry.RegisterContextMenuContributor()` 注册菜单项
3. **Context 传递** — 当前活动世界通过 `SetContext("activeWorld", world)` 传递给回调

## 公开 API

### SceneBrowserContextMenu（常量定义）

```csharp
public static class SceneBrowserContextMenu
{
    public const string BackgroundId = "scene_browser.background";  // 右键菜单 context ID
    public const string KeyActiveWorld = "activeWorld";              // 活动世界 context key
}
```

### 外部扩展示例

```csharp
// 方式一：注册贡献者（推荐，批量注册）
EditorMenuRegistry.RegisterContextMenuContributor(new MyContributor());

// 方式二：直接注册单个菜单项
EditorMenuRegistry.RegisterContextMenuItem(
    SceneBrowserContextMenu.BackgroundId,
    new EditorMenuItem("Add Entity/My Entity",
        Command: new EditorCommand { ... },
        SortOrder: 500));

// 方式三：注册动态回调
EditorMenuRegistry.RegisterContextMenuCallback(
    SceneBrowserContextMenu.BackgroundId,
    builder => builder.Add("Dynamic", () => { ... }));
```

## 已注册的实体类型

| 菜单项 | 模块 | 组件 | SortOrder |
|--------|------|------|-----------|
| Camera | Scene | NNCameraComponentData | 100 |
| Sprite | Scene | NNSpriteRendererComponentData | 200 |
| Audio Source | Media | NNAudioSourceComponentData | 300 |
| Video Player | Media | NNVideoPlayerComponentData | 400 |

## 文件清单

| 文件 | 模块 | 说明 |
|------|------|------|
| `SceneBrowserContextMenu.cs` | Scene/Public | context ID 常量 |
