# Neverness Engine — 最小可运行 2D 渲染管线 MVP 实施计划

> 目标：EditorViewport 能实时看到 Scene 中带有 Transform + SpriteRenderer 的 Entity。
> 原则：不做 RenderGraph / Material System / Shader Asset，硬编码 Builtin Shader，后续可无痛迁移到 Diligent Engine。

---

## 〇、当前已有的基础设施

### C++ Runtime

| 模块 | 关键内容 |
|------|----------|
| NNRuntimeScene | entt ECS、NNWorld、NNRuntimeScene、Transform/SpriteRenderer/Camera 组件、TransformSystem、CameraSystem、SystemScheduler |
| NNRuntimeRHI | VGFX 抽象接口（ITexture、IShader/IShaderProgram、IMesh）、OpenGL 后端（Texture2D、Shader、ShaderProgram、FrameBuffer、RenderTarget2D、Mesh） |
| NNNativeEngineAPI | C ABI 函数指针表（NNRenderAPI、NNSceneAPI、NNRenderAssetAPI 等），Handle 全部 uint64_t |
| NNRuntimeImGui | SDL3 + OpenGL3 后端的 ImGui 集成 |
| NNRuntimeRenderAssets | GPU 纹理资源管理 |

### C# Managed

| 模块 | 关键内容 |
|------|----------|
| Neverness.Runtime.Scene | SceneWorld、SceneEntity、SceneNativeBridge（薄封装 NNSceneAPI） |
| Neverness.Runtime.Interop | EngineNativeApiBootstrap（安装函数指针表） |
| Neverness.Editor.Scene | EditorViewport（空壳 OnGUI）、SceneBrowser、DetailInspector |

### 关键约束

- C ABI：`__stdcall`，函数指针表，不用 DllImport
- Handle 统一 `uint64_t`
- 组件 POD，无虚表
- 后端仅 OpenGL 3.3（GLAD）
- SDL3 窗口系统

## 一、架构总览

```
C# Editor                           C++ Runtime
─────────                           ───────────
EditorViewport                      NNRuntimeRenderer2D
  │                                   │
  ├─ ImGui.Begin("Viewport")          ├─ SceneRenderer
  │                                   │   │
  ├─ 调用 NativeAPI                   │   ├─ SpriteRenderSystem
  │   RenderSceneToTexture()          │   │   └─ Query<Transform, SpriteRenderer>
  │                                   │   │   └─ 生成 SpriteDrawCommand[]
  ├─ 收到 TextureId (GLuint)          │   │
  │                                   │   ├─ Renderer2D
  ├─ ImGui.Image(textureId, ...)      │   │   ├─ Bind Builtin Shader
  │                                   │   │   ├─ Set Camera MVP
  └─ ImGui.End()                      │   │   ├─ Batch Quad Draw
                                      │   │   └─ → RHI (OpenGL)
                                      │   │
                                      │   └─ FramebufferObject
                                      │       ├─ Color Texture
                                      │       └─ 返回 TextureId
                                      │
                                      └─ NNRuntimeRHI (已有)
                                          ├─ OpenGL::ShaderProgram
                                          ├─ OpenGL::FrameBuffer
                                          ├─ OpenGL::RenderTarget2D
                                          └─ OpenGL::Texture2D
```

### 核心流程

```
Scene.Update()
  → TransformSystem (已有，计算 WorldMatrix)
  → CameraSystem   (已有，计算 ProjectionMatrix)
  → SceneRenderer.Render(scene, camera, width, height)
      → SpriteRenderSystem.Collect(scene)
          → Query<Transform, SpriteRenderer>
          → 生成 SpriteDrawCommand[]（按 Layer/SortOrder 排序）
      → Renderer2D.Execute(commands, camera, framebuffer)
          → 绑定 Builtin Sprite Shader
          → 设置 Camera ViewProjection 矩阵
          → 遍历 DrawCommand
              → 设置 Transform 矩阵
              → 绑定 Texture
              → 设置 Color/UV/Blend
              → 绘制 Quad
      → 返回 Framebuffer Color Texture ID
```

### 3.1 SpriteDrawCommand

```cpp
// Public/SpriteDrawCommand.h
#pragma once
#include <cstdint>
#include <NNCore/Math/HMatrix.h>
#include <NNCore/Math/HFloat.h>

namespace NN::Runtime::Renderer2D
{
    /// 混合模式（与 NNSpriteRendererComponent::NNBlendMode 对齐）
    enum class BlendMode : uint32_t
    {
        Alpha = 0,
        Additive,
        Multiply,
        Opaque,
        Premultiplied
    };

    /// 单个 Sprite 绘制命令
    /// 全部 POD，无虚表，可直接 memcpy
    struct SpriteDrawCommand
    {
        NN::Core::matrix Transform;     // WorldMatrix（来自 TransformComponent）
        uint64_t          TextureHandle; // RHI Texture Handle（0 = 白色默认纹理）
        float             Color[4];      // RGBA tint
        float             UvRect[4];     // [u0, v0, u1, v1]
        uint32_t          Layer;         // 渲染层（用于排序）
        uint32_t          SortOrder;     // 层内排序
        BlendMode         Blend;         // 混合模式
        bool              FlipX;         // 水平翻转
        bool              FlipY;         // 垂直翻转
    };
}
```

### 4.1 SpriteRenderSystem

职责：遍历 ECS，收集 SpriteDrawCommand。

```cpp
// Public/SpriteRenderSystem.h

#include "SpriteDrawCommand.h"
#include <vector>

namespace NN::Runtime::Scene { class NNRuntimeScene; }

namespace NN::Runtime::Renderer2D
{
    /// 从 Scene 中收集所有可见 Sprite 的绘制命令
    class NN_RUNTIME_RENDERER2D_API SpriteRenderSystem
    {
    public:
        /// 收集帧绘制命令
        /// @param scene  场景引用
        /// @param outCommands 输出的绘制命令数组（每帧清空）
        void Collect(Scene::NNRuntimeScene& scene,
                     std::vector<SpriteDrawCommand>& outCommands);

    private:
        /// 对命令排序：先 Layer 升序，再 SortOrder 升序
        static void SortCommands(std::vector<SpriteDrawCommand>& commands);
    };
}
```

**实现要点（SpriteRenderSystem.cpp）：**

```cpp
void SpriteRenderSystem::Collect(
    Scene::NNRuntimeScene& scene,
    std::vector<SpriteDrawCommand>& outCommands)
{
    outCommands.clear();

    // entt 视图：查询同时拥有 Transform + SpriteRenderer 的实体
    auto view = scene.GetWorld().view<
        Scene::NNTransformComponent,
        Scene::NNSpriteRendererComponent>();

    for (auto [entity, transform, sprite] : view.each())
    {
        // 跳过不可见
        if (sprite.Flags & Scene::NNSpriteFlags::Visible == 0)
            continue;

        SpriteDrawCommand cmd{};
        cmd.Transform     = transform.WorldMatrix;
        cmd.TextureHandle = sprite.TextureAsset;
        cmd.Color[0]      = sprite.Color[0];
        cmd.Color[1]      = sprite.Color[1];
        cmd.Color[2]      = sprite.Color[2];
        cmd.Color[3]      = sprite.Color[3];
        cmd.UvRect[0]     = sprite.UvRect[0];
        cmd.UvRect[1]     = sprite.UvRect[1];
        cmd.UvRect[2]     = sprite.UvRect[2];
        cmd.UvRect[3]     = sprite.UvRect[3];
        cmd.Layer         = sprite.Layer;
        cmd.SortOrder     = sprite.SortOrder;
        cmd.Blend         = static_cast<BlendMode>(sprite.Blend);
        cmd.FlipX         = (sprite.Flags & Scene::NNSpriteFlags::FlipX) != 0;
        cmd.FlipY         = (sprite.Flags & Scene::NNSpriteFlags::FlipY) != 0;

        outCommands.push_back(cmd);
    }

    SortCommands(outCommands);
}

void SpriteRenderSystem::SortCommands(std::vector<SpriteDrawCommand>& commands)
{
    std::sort(commands.begin(), commands.end(),
        [](const SpriteDrawCommand& a, const SpriteDrawCommand& b)
        {
            if (a.Layer != b.Layer) return a.Layer < b.Layer;
            return a.SortOrder < b.SortOrder;
        });
}
```

**实现要点（Renderer2D.cpp）：**

```cpp
struct Renderer2D::Impl
    {
        // Builtin Shader
        uint32_t SpriteShaderProgram = 0;

        // Quad 几何体
        uint32_t QuadVAO = 0;
        uint32_t QuadVBO = 0;
        uint32_t QuadEBO = 0;

        // 白色默认纹理（无 Texture 时使用）
        uint32_t WhiteTexture = 0;

        // 当前相机
        CameraData CurrentCamera;

        // 当前混合模式
        BlendMode CurrentBlend = BlendMode::Alpha;

        // 统计
        uint32_t DrawCallCount = 0;
        uint32_t QuadCount = 0;
    };

    bool Renderer2D::Initialize()
    {
        m_Impl = new Impl();

        // 1. 编译 Builtin Shader（见第六节）
        m_Impl->SpriteShaderProgram = CompileBuiltinSpriteShader();

        // 2. 创建 Unit Quad VAO
        //    顶点: Position(x,y,z) + UV(u,v) → 5 floats
        //    {-0.5,-0.5, 0, 0, 0}
        //    { 0.5,-0.5, 0, 1, 0}
        //    { 0.5, 0.5, 0, 1, 1}
        //    {-0.5, 0.5, 0, 0, 1}
        //    索引: {0,1,2, 2,3,0}
        CreateQuadGeometry(m_Impl->QuadVAO, m_Impl->QuadVBO, m_Impl->QuadEBO);

        // 3. 创建 1x1 白色纹理
        m_Impl->WhiteTexture = CreateWhiteTexture1x1();

        return true;
    }

    void Renderer2D::BeginScene(const CameraData& camera, uint32_t width, uint32_t height)
    {
        m_Impl->CurrentCamera = camera;
        m_Impl->DrawCallCount = 0;
        m_Impl->QuadCount = 0;

        glViewport(0, 0, width, height);

        // 使用 Sprite Shader
        glUseProgram(m_Impl->SpriteShaderProgram);

        // 设置 ViewProjection 矩阵
        glUniformMatrix4fv(
            glGetUniformLocation(m_Impl->SpriteShaderProgram, "u_ViewProjection"),
            1, GL_FALSE, &camera.ViewProjectionMatrix.m[0][0]);

        // 绑定 Quad VAO
        glBindVertexArray(m_Impl->QuadVAO);
    }

    void Renderer2D::Submit(const std::vector<SpriteDrawCommand>& commands)
    {
        for (const auto& cmd : commands)
        {
            // 设置混合模式
            SetBlendMode(cmd.Blend);

            // 设置 Transform 矩阵
            glUniformMatrix4fv(
                glGetUniformLocation(m_Impl->SpriteShaderProgram, "u_Transform"),
                1, GL_FALSE, &cmd.Transform.m[0][0]);

            // 设置 Color
            glUniform4fv(
                glGetUniformLocation(m_Impl->SpriteShaderProgram, "u_Color"),
                1, cmd.Color);

            // 设置 UV Rect
            glUniform4fv(
                glGetUniformLocation(m_Impl->SpriteShaderProgram, "u_UvRect"),
                1, cmd.UvRect);

            // 设置 Flip
            glUniform1i(glGetUniformLocation(m_Impl->SpriteShaderProgram, "u_FlipX"),
                        cmd.FlipX ? 1 : 0);
            glUniform1i(glGetUniformLocation(m_Impl->SpriteShaderProgram, "u_FlipY"),
                        cmd.FlipY ? 1 : 0);

            // 绑定纹理
            BindTexture(cmd.TextureHandle);

            // 绘制 Quad
            glDrawElements(GL_TRIANGLES, 6, GL_UNSIGNED_INT, nullptr);

            m_Impl->DrawCallCount++;
            m_Impl->QuadCount++;
        }
    }

    void Renderer2D::BindTexture(uint64_t textureHandle)
    {
        uint32_t glTexId;
        if (textureHandle == 0)
        {
            // 无纹理 → 使用白色默认纹理
            glTexId = m_Impl->WhiteTexture;
        }
        else
        {
            // 从 RHI Texture Handle 获取 OpenGL Texture ID
            // 需要通过 RHI 的 Handle → Object 映射
            glTexId = ResolveTextureGLId(textureHandle);
        }

        glActiveTexture(GL_TEXTURE0);
        glBindTexture(GL_TEXTURE_2D, glTexId);
    }
```

**实现要点（SceneRenderer.cpp）：**

```cpp
uint32_t SceneRenderer::Render(Scene::NNRuntimeScene& scene,
                                uint32_t width, uint32_t height)
{
    // 1. 确保 Framebuffer 尺寸正确
    if (m_Framebuffer.GetWidth() != width ||
        m_Framebuffer.GetHeight() != height)
    {
        m_Framebuffer.Resize(width, height);
    }

    // 2. 获取相机数据
    CameraData camera = FindMainCamera(scene);

    // 3. 收集绘制命令
    m_SpriteSystem.Collect(scene, m_Commands);

    // 4. 绑定 FBO
    m_Framebuffer.Bind();

    // 5. 清屏
    glClearColor(0.1f, 0.1f, 0.1f, 1.0f);
    glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);

    // 6. 渲染
    m_Renderer.BeginScene(camera, width, height);
    m_Renderer.Submit(m_Commands);
    m_Renderer.EndScene();

    // 7. 解绑 FBO
    m_Framebuffer.Unbind();

    return m_Framebuffer.GetColorTextureId();
}

CameraData SceneRenderer::FindMainCamera(Scene::NNRuntimeScene& scene)
{
    CameraData data;

    // 查询带有 CameraComponent 的实体
    auto view = scene.GetWorld().view<
        Scene::NNTransformComponent,
        Scene::NNCameraComponent>();

    for (auto [entity, transform, camera] : view.each())
    {
        // 取第一个相机作为主相机
        data.ProjectionMatrix = camera.ProjectionMatrix;
        // View = 逆 WorldMatrix（简化：正交相机只需取逆平移）
        data.ViewMatrix = NN::Core::matrix::Inverse(transform.WorldMatrix);
        data.ViewProjectionMatrix = data.ProjectionMatrix * data.ViewMatrix;
        data.OrthoWidth  = camera.OrthoWidth;
        data.OrthoHeight = camera.OrthoHeight;
        data.Near = camera.Near;
        data.Far  = camera.Far;
        break; // MVP 阶段只取第一个相机
    }

    return data;
}
```

## 五、FramebufferObject 实现

```cpp
// Private/FramebufferObject.cpp
#include "FramebufferObject.h"
#include <glad/glad.h>

namespace NN::Runtime::Renderer2D
{
    bool FramebufferObject::Initialize(uint32_t width, uint32_t height)
    {
        m_Width = width;
        m_Height = height;

        // 创建 FBO
        glGenFramebuffers(1, &m_FBO);
        glBindFramebuffer(GL_FRAMEBUFFER, m_FBO);

        // 颜色附件：Texture2D
        glGenTextures(1, &m_ColorTextureId);
        glBindTexture(GL_TEXTURE_2D, m_ColorTextureId);
        glTexImage2D(GL_TEXTURE_2D, 0, GL_RGBA8,
                     width, height, 0,
                     GL_RGBA, GL_UNSIGNED_BYTE, nullptr);
        glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_LINEAR);
        glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_LINEAR);
        glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_S, GL_CLAMP_TO_EDGE);
        glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_T, GL_CLAMP_TO_EDGE);
        glFramebufferTexture2D(GL_FRAMEBUFFER, GL_COLOR_ATTACHMENT0,
                               GL_TEXTURE_2D, m_ColorTextureId, 0);

        // 深度附件：Renderbuffer
        glGenRenderbuffers(1, &m_DepthRenderbuffer);
        glBindRenderbuffer(GL_RENDERBUFFER, m_DepthRenderbuffer);
        glRenderbufferStorage(GL_RENDERBUFFER, GL_DEPTH24_STENCIL8,
                              width, height);
        glFramebufferRenderbuffer(GL_FRAMEBUFFER, GL_DEPTH_STENCIL_ATTACHMENT,
                                  GL_RENDERBUFFER, m_DepthRenderbuffer);

        // 完整性检查
        if (glCheckFramebufferStatus(GL_FRAMEBUFFER) != GL_FRAMEBUFFER_COMPLETE)
        {
            glBindFramebuffer(GL_FRAMEBUFFER, 0);
            return false;
        }

        glBindFramebuffer(GL_FRAMEBUFFER, 0);
        return true;
    }

    void FramebufferObject::Resize(uint32_t width, uint32_t height)
    {
        if (width == 0 || height == 0) return;
        Shutdown();
        Initialize(width, height);
    }

    void FramebufferObject::Bind()
    {
        glBindFramebuffer(GL_FRAMEBUFFER, m_FBO);
    }

    void FramebufferObject::Unbind()
    {
        glBindFramebuffer(GL_FRAMEBUFFER, 0);
    }

    void FramebufferObject::Shutdown()
    {
        if (m_DepthRenderbuffer) {
            glDeleteRenderbuffers(1, &m_DepthRenderbuffer);
            m_DepthRenderbuffer = 0;
        }
        if (m_ColorTextureId) {
            glDeleteTextures(1, &m_ColorTextureId);
            m_ColorTextureId = 0;
        }
        if (m_FBO) {
            glDeleteFramebuffers(1, &m_FBO);
            m_FBO = 0;
        }
    }

    FramebufferObject::~FramebufferObject()
    {
        Shutdown();
    }
}
```

## 六、Builtin Sprite Shader

硬编码在 `Public/BuiltinShaders.h` 中。

### Vertex Shader

```glsl
#version 330 core

layout(location = 0) in vec3 a_Position;
layout(location = 1) in vec2 a_TexCoord;

uniform mat4 u_ViewProjection;
uniform mat4 u_Transform;
uniform vec4 u_UvRect;   // [u0, v0, u1, v1]
uniform int  u_FlipX;
uniform int  u_FlipY;

out vec2 v_TexCoord;
out vec4 v_Color;

void main()
{
    gl_Position = u_ViewProjection * u_Transform * vec4(a_Position, 1.0);

    // UV 从 Rect 映射
    vec2 uv = a_TexCoord;
    uv = mix(u_UvRect.xy, u_UvRect.zw, uv);

    // Flip 支持
    if (u_FlipX == 1) uv.x = 1.0 - uv.x;
    if (u_FlipY == 1) uv.y = 1.0 - uv.y;

    v_TexCoord = uv;
}
```

### Fragment Shader

```glsl
#version 330 core

in vec2 v_TexCoord;

uniform sampler2D u_Texture;
uniform vec4      u_Color;  // Tint color

out vec4 FragColor;

void main()
{
    vec4 texColor = texture(u_Texture, v_TexCoord);
    FragColor = texColor * u_Color;
}
```

## 十一、RHI 分层约束

**严格禁止** 在 NNRuntimeRenderer2D 中直接暴露 `GLuint` 到高层。

当前 MVP 阶段的分层策略：

```
NNRuntimeRenderer2D
  ├── Renderer2D —— 内部直接调用 OpenGL（MVP 阶段可接受）
  ├── FramebufferObject —— 内部直接调用 OpenGL
  └── 对外接口 —— 只暴露 uint32_t TextureId，不暴露 GLuint 类型
```

**未来 Diligent 迁移时的改造点：**

| 当前 MVP | Diligent 迁移后 |
|----------|----------------|
| `glUseProgram()` | `IRenderer2D` 内部改为 Diligent `IPipelineState` |
| `glGenFramebuffers` | `FramebufferObject` 内部改为 Diligent `ITextureView` |
| `uint32_t TextureId` | 改为 `RHITextureHandle`（跨后端统一 Handle） |
| `glUniform*` | 改为 Diligent `IBuffer` + Map/Unmap |
| 硬编码 GLSL | 改为 Diligent Shader 预编译 |

**关键：外部调用者（SceneRenderer、EditorViewport）不需要任何改动。**
