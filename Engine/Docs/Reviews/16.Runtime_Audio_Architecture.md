# Runtime Audio 分层架构设计

> **状态：** 实施中
>
> **范围：** C# 音频播放抽象层 + C++ SDL3 backend + Editor AudioViewer 播放控制

---

## 架构总览

```
ImAudioViewerWindow (Editor.Media)      ← 消费者
    ↓
IAudioPlayer (Runtime.Audio)            ← C# 接口抽象
    ↓
NativeAudioPlayer (Runtime.Audio.Native)← C# P/Invoke 桥接
    ↓
NNAudio_* C ABI (NevernessRuntime-Media)← C 导出函数
    ↓
NN::Core::AudioPlayer (MediaCore)       ← SDL3 callback 播放
    ↓
SDL3 + FFmpeg                           ← Native backend
```

**原则：** SDL3 不出现在 C# 的任何文件中。Editor 只是 Runtime Audio 的消费者。

---

## 模块依赖

```
Editor.Media → Runtime.Audio (纯接口，零依赖)
                 ↑
Runtime.Audio.Native → Runtime.Audio
                 ↓
           NNAudio_* C ABI (NevernessRuntime-Media DLL)
                 ↓
           NevernessCore-MediaCore (SDL3 + FFmpeg)
```

## Phase A: C++ NNAudio_* C ABI

在 `NNRuntimeMedia` 模块中实现，桥接 `NN::Core::AudioPlayer`。

### C ABI 函数

```c
uint64_t NNAudio_CreatePlayer();
void     NNAudio_DestroyPlayer(uint64_t handle);
bool     NNAudio_LoadFile(uint64_t handle, const char* filePathUtf8);
void     NNAudio_Play(uint64_t handle);
void     NNAudio_Pause(uint64_t handle);
void     NNAudio_Resume(uint64_t handle);
void     NNAudio_Stop(uint64_t handle);
void     NNAudio_Seek(uint64_t handle, double seconds);
double   NNAudio_GetPlaybackTime(uint64_t handle);
double   NNAudio_GetDuration(uint64_t handle);
bool     NNAudio_IsPlaying(uint64_t handle);
void     NNAudio_SetVolume(uint64_t handle, float volume);
float    NNAudio_GetVolume(uint64_t handle);
void     NNAudio_SetLoop(uint64_t handle, bool loop);
```

### 内部实现

- `NNAudio_CreatePlayer()` → 创建 `NN::Core::AudioPlayer` 实例
- `NNAudio_LoadFile()` → 创建 `NN::Core::AudioClip`（VFS + FAudioDecoder）
- `NNAudio_Play()` → `AudioPlayer::Play()`（打开 SDL3 设备 + 流 + 回调）
- 其余函数直接委托 `IAudioPlayer` 接口

### 依赖变更

`NNRuntimeMedia/CMakeLists.txt` 新增链接：
```cmake
target_link_libraries(NevernessRuntime-Media PUBLIC
    NevernessCore-MediaCore    # AudioPlayer + FAudioDecoder + SDL3
    NevernessRuntime-VFS       # VirtualFileSystem
)
```

public interface IAudioPlayer : IDisposable

    bool Load(string filePath);

    void Resume();

## Phase C: C# Runtime.Audio.Native P/Invoke

`Engine/Source/Managed/Runtime/Neverness.Runtime.Audio.Native/`

### AudioNative.cs

```csharp
[LibraryImport("NevernessRuntime-Media")]
internal static partial ulong NNAudio_CreatePlayer();

[LibraryImport("NevernessRuntime-Media")]
internal static partial void NNAudio_DestroyPlayer(ulong handle);

// ... 其余 NNAudio_* 函数
```

### NativeAudioPlayer

实现 `IAudioPlayer`，持有 native handle，委托给 `AudioNative.NNAudio_*`。

### NativeAudioService

实现 `IAudioService`，工厂方法返回 `NativeAudioPlayer`。

## Phase D: Editor.Media 更新

### AudioAssetOpener

- DI 注入 `IAudioService`
- `CreatePlayer()` + `Load(sourceFilePath)`
- 传递 `IAudioPlayer` 给 ImAudioViewerWindow

### ImAudioViewerWindow

- Play/Pause 按钮
- 进度条（SliderFloat + Seek）
- 音量滑块
- Loop 复选框
- 播放时间显示
- 关闭时 Dispose player
